# Kindergarten

## The Big Idea

The universe might be like a big pattern.

A pattern is something that repeats.

Like:

- clap, clap, pause
- red, blue, red, blue
- little branches on a big tree

The Divine Blueprint asks:

```text
What if the universe has a pattern too?
```

## Copy With A Little Change

Some things copy themselves.

A plant can grow seeds.

A baby animal can look like its parents.

A song can have the same tune but a new verse.

The Divine Blueprint says reality might do something like that. It might make
new patterns that are like the old pattern, but not exactly the same.

That is how things can stay familiar and still change.

## What The Computer Check Does

A computer can help grown-ups check the idea.

It checks whether the rules fit together.

It asks:

```text
Do the rules agree?
Does the pattern stay steady?
Is the new pattern almost the same, but not exactly the same?
```

## Not Proven Yet

Not proven yet:

The computer check does not prove the universe really works this way.

It only checks that the idea fits together with itself.

Scientists still need to test it in the real world.

## Why It Feels Big

If the idea turns out to match nature, it could help explain why the universe
has patterns everywhere.
